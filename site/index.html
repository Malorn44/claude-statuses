    About this mirror

    status.claude.com publishes uptime per component but no single "any part of Claude is down" view. This site is that aggregate.

    We poll the public Statuspage, archive each incident timeline as JSON (in this repo's data/), and compute a single uptime number that treats any component being degraded as downtime.

    Not affiliated with Anthropic. Inspired by mrshu/github-statuses.
